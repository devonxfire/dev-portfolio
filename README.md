# dev-portfolio

Portfolio website built with React, TypeScript, Vite, and shadcn/ui.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack Query

## Port

Development server runs on `http://localhost:8080`
